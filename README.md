# File Organizer (Python)

## Description
A Python script that automatically organizes files in a folder based on their extension

## Features
 - Sorts files into categories
 - Automatically creates folders if not present
 - Beginner-friendly and easy to use
 - Works on Windows

## File Categories
 - **Images**: '.jpg' , '.jpeg' , '.png' , '.gif' , '.bmp' , '.webp' , '.tiff'
 - **Documents**: '.txt' , '.pdf' , '.docx' , '.xlsx' , '.doc' , 
 '.ppt' , '.pptx' , '.xls' , '.csv'
 - **Videos**: '.mp4' , ',mkv' , '.avi' , '.mov' , '.wmv' , '.flv'
 - **Audios**: '.mp3' , '.wav' , '.aac' , '.flac' , '.ogg'
 - **Archieves**: '.zip' , '.rar' , '.7z' , '.tar' , '.gz'
 - **Others**:  Remaining file types

## How It works
1. User enters a folder path
2. Script reads all files in the folder
3. Identiies fill extensions
4. moves files into related folders

## Technologies Used
 - Python
 - os module
 - shutil module
